全平台留电话-大数据版轰死它
关于抖音视频自动化给平台客服留电话的py脚本，原理其实很简单，用测试浏览器，结合py自动翻页，寻找页面中的固定元素，然后发送脚本中预设的文字即可。第一段脚本是在关键词中寻找带“广告”的链接。第二个脚本是读取链接并且发送预设文字。（脚本中有很多不完善的地方，欢迎大神们修改）

1228.py中说明：

EDGE_DRIVER_PATH = r"C:\Users\Administrator\Downloads\edgedriver_win64\msedgedriver.exe"是调取msedgedriver.exe的路径，可以打开edge浏览器自己搜索msedgedriver.exe下载即可，然后替换路径。

KEYWORDS = ["我好帅"]这句是用于跳过人机验证的，这里的文字你可以随便改。更复杂的跳验证方式懒得弄。

AD_KEYWORDS = ["无痛人流", "小本加盟", "IT培训", "免费领pos机", "学历提升", "婚恋网", "装修", "房产中介", "成人教育",
               "公考机构", "考研机构", "办理信用卡"]这里是你想加入的行业，直接加即可，注意英文的逗号。

别的没有了，注意脚本没有进行封装（我不会）所以需要安装对应的py库和环境，不会的自己百度或者gpt一下。

12282.py中说明：
在前一段脚本中，爬虫会收集搜索引擎中公开的广告链接，包括百度的竞价广告，并且把他们保持到一个txt文件中，这一步的脚本会挨个读取txt文本的内容，并且自动打开留下脚本中预设的文字。
把脚本中的
BUTTON_CLICK_WAIT_SECONDS = 10  # 按钮点击后等待时间
MESSAGES_TO_SEND = [
    "你好",
    "我想要了解一下",
    "手机号码：13XXXXXXXXX",
    "谢谢",
]
手机号替换即可。
经过试验，每次抓取的广告数量大概是40-60条左右，其中有非竞价广告，淘宝广告，注册广告等，有效广告可以达到2/3.而且电话打的很迅速，基本隔天就会有销售联系你，加你微信，这种方法对于靓号比如尾号5678，尾号8888等非常有效，这个号基本废了。淘宝广告不生效，注册广告不生效。
